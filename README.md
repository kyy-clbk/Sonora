### ✨ Fitur Utama
* 🌸 **Desain Estetik:** Tampilan UI modern dan bersih (Clean UI).
* 🌊 **Fluid Player:** Animasi usap (swipe) pemutar musik ala Apple Music.
* 📂 **Manajemen Playlist:** Buat daftar putar dan tambahkan cover kustom (dari galeri HP).
* 🎛️ **Kontrol Penuh:** Mode *Shuffle*, *Play/Pause*, *Next/Prev*, dan slider durasi akurat.
* 🎵 **Deteksi Otomatis:** Otomatis membaca semua lagu (MP3/FLAC) beserta sampul album (Album Art) dari penyimpanan HP.
